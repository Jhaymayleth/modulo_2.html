# Inventory Management System (Python)

## Description
This project is a basic **inventory management system** developed in Python.  
It allows users to add products, view the inventory, and calculate basic statistics through a console-based menu.

The system applies fundamental programming concepts such as:
- Lists and dictionaries
- Functions and modular logic
- Input validation
- Control flow (loops and conditionals)

---

## Features

-  Add products with validation (name, price, quantity)
-  Display all products in a clear format
-  Calculate inventory statistics:
  - Total inventory value
  - Total number of units
-  Interactive console menu
-  Error handling for invalid inputs
