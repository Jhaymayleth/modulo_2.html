# Python Console Applications Collection

##  Overview
This repository contains a collection of **console-based Python applications** focused on inventory management and basic shopping systems.  

Each project demonstrates fundamental programming concepts such as:
- Data structures (lists, dictionaries)
- Modular programming
- Input validation
- File handling (CSV)
- Control flow and user interaction

These projects are designed as learning exercises and practical implementations of real-world logic.

---

## Projects Included

###  1. Inventory Management System (Basic)
A simple inventory system that allows users to:
- Add products
- Display inventory
- Calculate statistics

**Key concepts:**
- Lists and dictionaries  
- Functions  
- Input validation  
- Loops and conditionals  

---

### 💾 2. Inventory Management System with CSV
An enhanced version of the inventory system with data persistence.

**Features:**
- Full CRUD operations (Create, Read, Update, Delete)
- Inventory statistics
- Save and load data using CSV files
- Error handling and validation
- Modular structure (`app.py`, `servicios.py`, `archivos.py`)

**Use case:**  
Simulates a more realistic system where data is stored and reused.

---

### 🛒 3. Riwistore Shopping Cart
A command-line shopping cart system that allows users to calculate the total cost of multiple products.

**Features:**
- Add multiple products dynamically
- Calculate subtotal and total cost
- Continuous input loop
- External validation module usage

**Logic highlights:**
- Iterative product entry
- Accumulated totals
- Modular validation
